# MarkDown
> MarkDown是一种轻量级标记语言，markdown内可以直接使用html语法

## 一、基本语法

### 1.标题
```text
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 2.引用
```text
> 引用可嵌套使用
```

> markdown文件后缀为md

### 3.字体
```text
*加粗斜体*
**加粗**
***斜体加粗***
==高亮==
~~删除~~  
<u>下划线</u>
```

*斜体*  
**加粗**  
***斜体加粗***  
==高亮==  
~~删除~~  
<u>下划线</u>

### 4.段落
```text
换行：两个以上的空格后回车
这是一  
条语句。

段落：空一行
这是第一段。

这是第二段。
```

这是一  
条语句。

这是第一段。

这是第二段。

### 5.分割线
```text
三个或更多连续的星号
***
```

***

### 6.列表
```text
1. 有序列表
2. 有序列表
3. 无序列表
    - 列表嵌套
    - 列表嵌套
4. 待办列表
    - [ ] 代办项
    - [ ] 代办项
```

1. 有序列表
2. 有序列表
3. 无序列表
    - 列表嵌套
    - 列表嵌套
4. 待办列表
    - [ ] 代办项
    - [x] 代办项

### 7.脚注
```text
脚注标记[^n]

脚注解释[^n]:
```

关于健康[^1]饮食。

### 8.代码块
```text
行内代码`main`

代码块
[占位无意义]```C
[占位无意义]int main() {
[占位无意义]    return 0;
[占位无意义]}
[占位无意义]```
```

C语言的可执行程序入口是`main`函数。

```C
int main() {
    return 0;
}
```

### 9.超链接
```text
行内式：[链接名](链接地址)
引用式：[链接名][标记，类似脚注]
```

[Rust官网](https://rust-lang.org/)
[Rust官网][a]

### 10.图片
```text
行内式：![图片描述](图片地址)
引用式：![图片描述][标记，类似脚注]

图片可以嵌套在超链接中
```

![头像](image/test.jpg)
![头像][b]
